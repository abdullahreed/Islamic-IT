# Islamic-IT

An investigation into how submission to Almighty God can best assist the IT industry.

It is an html file that can be shared with Muslim developers and IT users.

"Technology Reminder" is not intended for use in preaching as it is more suitable for a Muslim audience. Be sure to use wisdom when preaching.

For people who already believe in God, He, says: "Therefore do remind, surely reminding does profit."
